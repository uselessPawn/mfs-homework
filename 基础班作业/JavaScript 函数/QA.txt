问答题
1. 为什么会有函数？函数有什么用？
因为会面临相同一段代码写多次的情况。

函数能让一段具有完整功能代码可以重复使用。

2. 有哪几种声明函数的方式？最常用的是哪些？
构造函数法：
var myFun = new Function("a", "b", "return a * b");

函数声明法：
function myFunction(a, b) {
    return a * b;
}

函数表达式法：
var myFun = function (a, b) {return a * b};

最常用的是函数声明法，函数表达式法次之，构造函数法基本不用。


3. 什么是函数返回值？如何设置函数返回值？如何获得函数返回值？
函数返回值就是函数执行完后的结果。
在函数内部使用return设置函数返回值。
调用函数并赋值给变量就能获得函数返回值。


4. 函数中 return 后面的语句会执行吗？为什么？
会。


5. 什么是函数提升？为什么会有函数提升？
提升（Hoisting）是 JavaScript 默认将声明提升到文件开始处的的行为。

好处：
函数提升允许在声明前使用，支持把函数写在文件后部分，便于阅读。
将函数声明提升到开头，比较适合解释器分析程序。
先解析函数声明便于作用域的关联。
