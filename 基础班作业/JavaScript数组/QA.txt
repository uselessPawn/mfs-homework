问答题
1. 如何声明/创建数组？
构造函数： var arr = new Array();
字面量：var arr =[]


2. 数组可以像对象一样插入属性吗？插入的属性会影响数组的 length 吗？如 a["sss"]="sss"
可以。因为数组本质上也是一个对象。但不会增加数组的length。


3. 如果访问数组中不存在的元素，会怎么样？
访问index不存在的元素的时候返回undefined。


4. join()函数返回值是什么？它可以完成什么到什么的转换？
返回的是一个字符串。它可以完成数组到字符串的转换。


5. 什么是栈？如何使用数组模拟栈？
栈是一个数据先进后出的数据结构。
输入数据用数组的push方法，
输出则是使用数组的pop方法。


6. 什么是队列？如何使用数组模拟队列？
队列是一个数据先进先出的数据结构。
输入数据用数组的push方法。
输出数据则是用数组的shift方法。


7. 为什么说 splice() 函数是数组修改的神器？它有哪些参数？各是什么含义？
因为splice可以完成数组的增删改。

index	必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。
howmany	必需。要删除的项目数量。如果设置为 0，则不会删除项目。
item1, ..., itemX	可选。向数组添加的新项目。

代码题：
请解释如下代码输出为什么结果不对，并请修改代码实现返回正确的结果。

```
var a=new Array(7,8,9,10,11);
a.sort();
console.log(a);// 输出 [10, 11, 7, 8, 9]
```
错误原因在于数组的sort函数默认的排序是按照ASCII码排序。修改方法：传入需要的比较函数。

修改后的代码：
```
 var a = new Array(7, 8, 9, 10, 11);
        a.sort((a, b) => {
            return a - b
        });

console.log(a); // 输出 [7, 8, 9, 10, 11]
```
