问答题：
1. CSS 中颜色有几种表示方法？RGB 表示法参数为数值和百分数时分别为什么意思？RGBA 表示法中 A 参数表示什么意思，取值范围是什么？
有命名颜色表示法，RGB函数表示法，RGBA函数表示法，十六进制表示法

数值是单色的强度，取值范围0-255。百分数表示百分比下的颜色强度，取值范围是0-100%
A表示透明度，取值范围是0.0-1.0，

2. 什么是前景色，什么是背景色？
前景色指元素，如字体的颜色。背景色则是元素里除前景色之外的颜色。

3. background 属性可以缩写哪些子属性(如:background-color)？他们的参数各是什么，有何含义？
background-color： 
背景颜色，参数一般可以是
#ffffff | rgb(255,255,255) | rgb(100%,100%,100%) |rgba(255,255,255,1)

background-image：
把图像设置为背景。
参数为url([资源的url])

background-repeat：
当图片大小不足时是否重复
参数为：
no-repeat 不重复
repeat-x 横向重复
repeat-y 纵向重复
repeat 横纵都重复

background-attachment：
背景图片是否固定或者随着页面的其余部分滚动而滚动
参数有
scroll: 背景图片随着页面的滚动而滚动，这是默认的
fixed: 背景图片不会随着页面的滚动而滚动
local: 背景图片会随着元素的滚动而滚动

background-position：
背景图片起始位置，默认左上角
参数：
x y 

%x %y
·起始位置的横坐标 =（元素宽度 - 图片宽度）* x%
·起始位置的纵坐标 = （元素高度 - 图片高度） * y%

[top|center|bottem] [left|center|right]


background-size：
背景图片大小
参数为：
1. 确切值，即具体的像素值，如果只设置一个值，那么第二值就变为auto
2. 百分比值：以父元素为基础值来设置背景图片的宽高。同样如果只设置一个值，那么第二个值就变为auto。 
3. cover：把背景图片扩展到足够大，以使背景图片能够完全覆盖背景区域
4. contain：把背景图片扩展到最大尺寸，以使得宽度和高度能够完全适应内容区域
cover和contain都是按比例扩大图片，区别在于cover在填满背景之前不会停止扩大，多出的部分会被裁剪。
而cantain则是扩大到边缘，哪怕未填补完背景也会停止，所以可能会有留白。   


4. 同时指定 background-color 和 background-image 时会怎么样？请分各种情况讨论
如果background-image是png格式的图片话，background-color就会在png图片中透明的部分生效；
如果不是png，那么background-color就无法生效。

5. background-position 可以写哪几种参数？他们的含义是什么？
可以写五种参数，分别为color url repeat attachment position
含义为：颜色 图片URL 重复与否 固定与否 起始位置


6. 什么是雪碧图？为什么要使用雪碧图？雪碧图的实现原理是什么？
即CSS sprite。
使用原因：
网站需要用到许多图片，如果分开申请，那就需要用很多HTTP申请，速度会很慢。
但如果只有一个图片，那就只需要一个HTTP链接。
同时HTML是懒加载机制，雪碧图能保证一些图片在还没加载到的时候不加载。这能让网站更快。

实现原理：
用background-positon改变图片的起始显示位置，保证每次只显示图片的一部分。

7. 现代前端中为什么很少使用雪碧图了？有哪些替代方案？相比之下有何好处？
原因一：http2的连接可以在一个连接中传输多个图片，所以雪碧图减少http链接的意义没了。
原因二：一些打包工具可以将图片打包成HTML元素，伙同HTML文件一起传输到客户端上，不需要多次请求。
原因三：JS可以代替雪碧图实现懒加载机制

好处：相比之下可以避免出现很大的雪碧图，导致请求一张过大的图片导致网站变慢。
