12. == 与 === 有何异同？
同：都可以进行相等性比较。

不同：

等号没有这样的性质。

代码题
1. 下面代码输出什么，为什么？
```
false && console.log("out")
```

什么都不输出。
因为在解析到 false && 的时候，整个逻辑判断的结果已经是false了，因而后面的console.log("out")不被执行

2. 下面代码输出什么？为什么？

```
console.log(undefined || null || "out")
```

输出out，"out"在逻辑判断时，等价于true，但没有类型强制转换成true。


3. 如下代码中，a 最终为多少？
```
var a = (1,2,3)
```
最终为3

如果改为如下代码那？为什么？

```
a = 1, 2, 3
```

为1。
因为解析的优先级为：(a = 1),(2),(3)
